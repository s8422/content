Сбор и анализ информации о томах {{< tag "LVM" >}}.

<!--more-->

## PV

- Показать краткую информацию о физических томах:

```bash
pvs
```

- Показать полную информацию о физических томах:

```bash
pvdisplay
```

- Показать полную информацию о физическом томе `/dev/sdb`:

```bash
pvdisplay '/dev/sdb'
```

## VG

- Показать краткую информацию о группах логических томов:

```bash
vgs
```

- Показать полную информацию о группах логических томов:

```bash
vgdisplay
```

- Показать полную информацию о группе логических томов `/dev/data/documents`:

```bash
vgdisplay 'data'
```

## LV

- Показать краткую информацию о логических томах:

```bash
lvs
```

- Показать полную информацию о логических томах:

```bash
lvdisplay
```

- Показать полную информацию о логическом томе `/dev/data/documents`:

```bash
lvdisplay '/dev/data/documents'
```
